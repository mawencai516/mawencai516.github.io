定义画布的基本属性

属性 描述 属性值(NaN为自适应，其他值为举例)
initialContentAlignment 画布初始位置 go.Spot.Center
allowMove 禁止移动节点 false
allowCopy 禁止复制 false
allowDelete 禁止删除 false
allowSelect 禁止选中 false
allowZoom 禁止缩放 false
allowHorizontalScroll 禁止水平滚动条 false
allowVerticalScroll 禁止垂直滚动条 false
isReadOnly 只读 true
padding 画布边距padding padding:80
margin 画布边距Margin new go.Margin(2, 0)或new go.Margin(1, 0, 0, 1)

公用属性

所有节点块定义都能通用的属性

属性 描述 属性值(NaN为自适应，其他值为举例)
minSize 最小范围 new go.Size(50, NaN)或new go.Size(50, 200)
maxSize 最大范围 同上
width 宽
height 高
margin 外边距 0或new go.Margin(2, 0)或new go.Margin(1, 0, 0, 1)
alignment 元素位置 go.Spot.BottomLeft 左下 go.Spot.BottomRight 右下 go.Spot.TopLeft 左上 go.Spot.TopRight 右上 go.Spot.Left go.Spot.Right go.Spot.Top go.Spot.Bottom
defaultAlignment 子元素位置 go.Spot.TopRight 右上
visible 设置是元素是否可见 false为不可见

go.Part 部分属性

属性 描述 属性值(NaN为自适应，其他值为举例)
zOrder:999 节点显示层级，越大越上面 默认NaN，按创建顺序排序，最先创建的在最底部
deletable:false 可删除的 默认true,false
movable:false 可移动的 默认true,false
selectable: false 可选择的 默认true,false
selectionAdorned:false 显示选中边框 默认true,false
isShadowed:true 显示阴影 默认false,true
shadowColor : “#ffda92” 阴影的颜色 默认gray
shadowOffset:new go.Point(3, 3) 阴影的位置偏移 默认为new go.Point(6, 6)
copyable:false 可复制的 默认true,false
isSelected:true 选中 默认true,false

go.Panels(面板元素)

  • 定义子元素在面板的的位置

| Auto | 子元素叠加布局 |
| Vertical | 子元素垂直布局 |
| Horizontal | 子元素水平布局 |
| Spot | 子元素Spot布局 |
| Position | 子元素坐标布局 |
| Table | 子元素表格布局 |

go.TextBlock(文本框元素)

属性 描述 属性值(NaN为自适应，其他值为举例)
stroke 文字颜色（边框颜色） “#333333”或”red”
font 字体 “10pt helvetica, arial, sans-serif”
editable 文本可编辑 默认true,false
textAlign 文本位置
minSize 文本域最小范围，设置范围后才能控制换行和裁切
maxSize 文本域最大范围
maxLines 文本显示的最大行数
isMultiline 允许换行 默认true,false
overflow 文本超出范围处理方式,设置文本域范围后该属性才会生效 go.TextBlock.OverflowEllipsis 显示缩略号

go.TextBlock.OverflowClip 裁切 |
